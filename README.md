# Premier League Players Performance Analysis (2024/25)

This project explores the performance of **562 Premier League players** using over **50 statistical features**. The analysis focuses on player efficiency, positional comparisons, and key performance indicators through data cleaning, exploratory analysis, feature engineering, and interactive visualizations.
