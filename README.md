# premier-league-2025-player-performance-analysis7

A complete exploratory data analysis (EDA) of Premier League players during the 2024/25 season using Python.

This project explores player performance through offensive, defensive, passing, possession, and goalkeeping statistics to identify key trends, standout performers, and relationships between metrics.

---

## 📊 Dataset

The dataset contains:

- **562 Premier League players**
- **53 statistical features**
- No missing values
- Offensive, defensive, passing, possession, goalkeeping and disciplinary statistics

Examples of available metrics:

- Goals
- Assists
- Expected Goals (xG)
- Shots on Target
- Tackles
- Interceptions
- Pass Accuracy
- Progressive Carries
- Saves
- Yellow Cards
- Red Cards

---

## 🎯 Objectives

The main objectives of this project are:

- Perform Exploratory Data Analysis (EDA)
- Understand relationships between player statistics
- Identify top-performing players
- Detect statistical patterns and correlations
- Create insightful visualizations
- Build a clean data analysis workflow

## 🚀 Key Insights

Some examples of insights obtained:

- Strong relationship between shots on target and goals.
- High xG does not always translate into high goal conversion.
- Midfielders dominate passing statistics.
- Defenders lead aerial duels and interceptions.
- Goalkeepers show significant differences in save percentages.
